My research theme is to install a telescope using MALTA2, a type of MAPS used in the ATLAS experiment, as a reference plane to facilitate beam tests at KEK (High Energy Accelerator Research Organization).

Construction of Beam Telescope at KEK

MALTA2 Beam Test Hardware Setup
現在取り組んでいるメインプロジェクトは、KEKにおけるビームテレスコープの設置です。 目標は、ユーザーが自分のDUT（Device Under Test）を持ち込むだけで、すぐにビームテストを行える環境を構築することです。 My current main project is setting up a beam test telescope at KEK. The goal is to create a user-friendly environment where users only need to bring their DUT (Device Under Test) to immediately perform beam tests.
Reference PlaneとしてMALTA2センサを使用することで、高精度なトラッキングを実現します。精密なアライメント調整と読み出しシステムの統合を行っています。 We utilize the MALTA2 sensor as a reference plane to achieve high-precision tracking. This setup requires precise mechanical alignment and integration of the readout systems.

What is MALTA2?

MAPS ATLAS CMOS Sensor
本研究の核となるMALTA2は、CERNのATLAS実験などの将来的なアップグレードに向けて開発された、Monolithic Active Pixel Sensor（MAPS）の一種です。 MALTA2, the core of this research, is a type of Monolithic Active Pixel Sensor (MAPS) developed for future upgrades of experiments like ATLAS at CERN.
センサーと読み出し回路が一体化されており、高い放射線耐性と高速な読み出し性能を兼ね備えています。これを参照面に用いることで、高レート環境下でも正確な飛跡検出が可能になります。 Unlike traditional detectors, the sensor and readout circuitry are integrated. It combines high radiation hardness with fast readout capabilities, enabling accurate tracking even in high-rate environments.

Simulation Study with Allpix Squared

Allpix² Corryvreckan Geant4
ハードウェアのセットアップと並行して、Allpix Squaredを用いたモンテカルロシミュレーションも行っています。Geant4ベースの計算で、検出器の応答や分解能を評価しています。 In parallel with hardware setup, I perform Monte Carlo simulations using Allpix Squared. I evaluate detector response and resolution using Geant4-based calculations.
また、再構成フレームワークであるCorryvreckanを使用し、実際のビームテストに向けた最適なジオメトリを探しています。 I also use the reconstruction framework Corryvreckan to find the optimal geometry for upcoming beam tests and to study algorithms for distinguishing scattered signals.
